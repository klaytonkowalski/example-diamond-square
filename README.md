## Controls

| Control | Description |
| ------- | ----------- |
| Left Click | Generate a new map. |
| Right Click | Scroll across the map. |
| 1 | Use grayscale graphics. |
| 2 | Use colored graphics. |
| 3 | Decrease the random scalar. |
| 4 | Increase the random scalar. |
